# 👨‍👩‍👧‍👦 Site Famille

Site familial et éducatif de la famille Dardeau — installable en PWA.

## Contenu

- **Dashboards enfants** : `dashboard_leana.html`, `dashboard_mathias.html`
- **Jeux éducatifs** : corps humain 3D avec quêtes vocales (`jeu_corps_humain.html`),
  phonétique (`phonetique_leana.html`), géométrie (`geometrie_mathias.html`),
  autres jeux dans `jeux/`
- **Créatif** : karaoké (`karaoke_leana.html`), studio avatar (`studio_avatar.html`),
  maestro (`maestro_jeux.html`)

## PWA

`manifest.json` + `sw.js` repris du prototype *famille-site* (v1, archivé).
Le système de login v1 est conservé dans `js/famille-app-v1.js` pour intégration future.

## Historique

Successeur du repo privé `famille-site` (archivé en juillet 2026).
